## Overview

Context menus expose common actions on plugin layers through right-click shortcuts. They mirror the toolbar but scope their actions to the clicked feature or layer.

## Layer-level actions

- **Zoom to TIN** — fit the view to the mesh extent.
- **Check cell angles** — flag cells with sliver triangles.
- **Reorder database** — normalize attribute-table IDs.
- **Wind mesh** — regenerate mesh in a sweep around a central point.

## Feature-level actions

- **Edit hydrograph** — opens the inflow/outflow time-series editor.
- **Apply rating table** — attach a Q-H curve to a feature.
- **Open boundary-condition dialog** — parametric editor for the clicked BC feature.

## Tips

- Right-clicking is often faster than hunting through the toolbar — learn the common shortcuts for the actions you use most.
- Feature-level menus only appear if the layer is in a compatible schema; check the template-layer creator if an expected action is missing.
