## Overview

The **Cross Sections** toolbar button opens a profile viewer that plots water-surface elevation and ground elevation along user-drawn transects, through time.

## Dialog window

![Cross Sections dialog](img/cross-sections-dialog.png)

## Dialog controls

| Control | Purpose |
| --- | --- |
| *Transect layer* | Line layer defining cross-section locations. |
| *Result file* | `.OUT` file containing WSE time series. |
| *Time step* | Snapshot to plot. |
| *Animate* | Steps through all timesteps. |

## Workflow

1. Draw cross-section lines on the map.
2. Open **Cross Sections**, pick the transect layer and result file.
3. The viewer plots ground and WSE along each transect.
4. Scrub through time or export plots.

## Tips

- Keep transects perpendicular to flow direction for meaningful hydraulic interpretation.
- Combine multiple transects on one figure to compare conveyance at different river stations.
